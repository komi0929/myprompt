# 開発サーバー起動ワークフロー

**ローカル開発のクイックスタート**

## Steps

// turbo-all

1. 依存関係をインストール:

```powershell
npm install
```

2. 開発サーバーを起動:

```powershell
npm run dev
```

3. ブラウザで http://localhost:3000 を開く

## Optional: Supabase ローカル起動

Supabaseローカル開発を使用する場合:

```powershell
npx supabase start
npm run dev
```
